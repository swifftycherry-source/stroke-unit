🤖 คำตอบ

📊 ตั้งค่าเชื่อมต่อ Google Sheet
ขั้นตอนการตั้งค่า:
1. เปิด Google Sheet ที่ต้องการ → Extensions → Apps Script
2. วางโค้ดจากหน้า "บันทึกข้อมูล" ลงใน Script Editor
3. แทน YOUR_SHEET_ID ด้วย ID ของ Sheet คุณ
4. Deploy → New Deployment → Web app
5. Execute as: Me | Who has access: Anyone → Deploy
6. คัดลอก Web app URL มาวางด้านบน
